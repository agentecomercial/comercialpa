📜 Setup das IAs (Claude.ai · ChatGPT · Gemini)

Configure cada IA UMA VEZ pra deixar o agente mais rápido. Depois é só clicar nos botões coloridos no resultado pra abrir cada uma com o prompt no clipboard.

🤖 Claude.ai (Anthropic) — recomendado

O Claude tem o melhor recurso para esse uso: Projetos. Você cola o System Prompt uma vez e todas as conversas no Projeto já usam ele.

1
Acesse claude.ai e faça login (gratuito).
2
Menu lateral → "Projetos" → "Novo Projeto". Nome: Agente Comercial Febracis.
3
Em "Instruções do Projeto", cole o System Prompt — Análise de Lead (aba "📜 Prompts" acima).
4
(Opcional) Crie outro Projeto Treinamento Febracis para o modo treino.
5
Volte aqui em ⚙️ Config → defina Claude.ai = Projeto configurado.

💬 ChatGPT (OpenAI)

O ChatGPT tem Custom GPTs (similar a Projetos). É preciso assinatura ChatGPT Plus para criar. Sem assinatura, use modo Conversa Avulsa.

Opção A — Custom GPT (precisa Plus)

1
Acesse chat.openai.com/gpts/editor.
2
Aba "Configure". Nome: Agente Comercial Febracis.
3
Em "Instructions", cole o System Prompt — Análise de Lead (aba "📜 Prompts").
4
Salve. Use sempre esse GPT pelo link chat.openai.com/g/SEU-GPT.
5
Em ⚙️ Config: ChatGPT = Custom GPT configurado.

Opção B — Conversa Avulsa (gratuito)

Não precisa configurar nada. Quando clicar no botão 💬 ChatGPT, o agente envia o System Prompt + contexto + fala juntos. Funciona em qualquer conversa nova.

Em ⚙️ Config: ChatGPT = Conversa Avulsa (completo) — já é o padrão.

✨ Gemini (Google)

O Gemini tem Gems (similar a Projetos/GPTs). Disponível no plano gratuito e no Gemini Advanced.

Opção A — Gem customizada

1
Acesse gemini.google.com/gems/create.
2
Nome: Agente Comercial Febracis.
3
Em "Instruções", cole o System Prompt — Análise de Lead (aba "📜 Prompts").
4
Salve. Aparece na sua lista de Gems.
5
Em ⚙️ Config: Gemini = Gem configurado.

Opção B — Conversa Avulsa

Sem configurar nada, o botão ✨ Gemini envia tudo junto (System Prompt + contexto). Em ⚙️ Config: Gemini = Conversa Avulsa — padrão.

Os System Prompts mestres. Use o de "Análise" para o Projeto/GPT/Gem principal. Use o de "Treinamento" para um segundo Projeto/GPT/Gem se quiser separar.
