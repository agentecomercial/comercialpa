💡 Como vou implementar:
Vou usar uma variável CSS --icv-scale no modal. Tudo dentro multiplica por ela. Mudar de uma escala pra outra é só trocar o valor (ex: 1.0 → 1.20). Posso até adicionar um botão de zoom no header do modal pra você ajustar ao vivo se quiser.
